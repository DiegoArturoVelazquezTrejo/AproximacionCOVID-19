Algoritmo Genético para aproximar datos COVID-19
========================

Objetivo: diseñar un algoritmo genético para optimizar los parámetros alfa, beta y mu
de la ecuación      alpha * e * -((x - mu)°2)/2
minimizando el error cuadrático medio.

### Fecha Mayo 16, 2020

### Para ejecutar:
```
$ python3 main.py

```
